< !DOCTYPE html>

Deklarasi merupakan elemen penting di awal dokumen HTML. Ia digunakan untuk menentukan tipe dokumen HTML yang akan digunakan. Misalnya, untuk HTML5

< html> < /html>

Tag < html> adalah salah satu tag yang paling fundamental dalam HTML. Ini menandakan awal dari sebuah dokumen HTML dan berisi semua elemen yang akan ditampilkan di browser. Setiap halaman web harus dimulai dengan tag < html> dan diakhiri dengan tag < /html>.

< head> < /head>

Tag head> adalah salah satu elemen yang penting dalam sebuah dokumen HTML. Ini digunakan untuk menyediakan informasi meta-data tentang dokumen HTML, seperti judul halaman, tautan ke stylesheet eksternal, skrip JavaScript, meta tag, dan banyak lagi.

< title> < /title>

Tag title> dalam HTML digunakan untuk menentukan judul dari dokumen HTML tersebut. Judul ini ditampilkan di title bar atau tab browser ketika halaman web itu dibuka. Contoh penggunaannya adalah sebagai berikut:

< body> < /body>

Tag body> dalam HTML digunakan untuk menampung semua konten yang akan ditampilkan di halaman web. Ini mencakup teks, gambar, tautan, elemen formulir, video, dan segala jenis konten lainnya yang ingin Anda tampilkan kepada pengguna.
