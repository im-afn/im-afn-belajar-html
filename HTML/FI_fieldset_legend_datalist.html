Element Fieldset

Digunakan untuk mengelompokkan elemen-elemen terkait dalam sebuah formulir, dan sering dipasangkan dengan tag < legend>, yang memberikan judul untuk < fieldset>. Ini membantu dalam mengorganisir dan membangun struktur formulir, terutama ketika ada beberapa input yang saling terkait. Membantu dalam pengelompokkan dan penyusunan yang lebih baik dari elemen-elemen formulir, meningkatkan pengalaman pengguna dan memudahkan pengembangan dan pemeliharaan halaman web.

Attribut fieldset

  • < fieldset disabled>< /fieldset>

digunakan untuk menonaktifkan atau mengunci semua elemen dalam grup < fieldset>, termasuk elemen-elemen formulir di dalamnya. Ini berguna ketika Anda ingin mengizinkan atau melarang pengguna untuk mengubah atau memasukkan data ke dalam grup formulir tertentu.

Personalia:







  • < form id="form1" action="/submit_form1">
    < fieldset form="form1">

    < -- isi -->

    < /fieldset>
    < /form>

Digunakan untuk menentukan formulir mana yang akan mengelola data yang dikirimkan oleh elemen-elemen yang berada di dalam < fieldset>. Ini berguna ketika Anda ingin mengelompokkan elemen formulir dalam sebuah < fieldset> tetapi ingin data yang dikirim oleh elemen-elemen tersebut ditangani oleh formulir yang berbeda.

Informasi Alamat



  • < fieldset data-name="contact">< /fieldset>

Dalam HTML, tag fieldset tidak memiliki atribut name. Ini karena elemen fieldset digunakan untuk mengelompokkan elemen-elemen formulir terkait dalam satu unit logis, dan biasanya tidak perlu diberi nama. Namun, jika Anda ingin menambahkan metadata tambahan ke dalam fieldset, Anda dapat menggunakan atribut data-*, yang memungkinkan Anda menambahkan data kustom ke elemen HTML tanpa mempengaruhi perilaku atau penampilan default. Element Legend

Dalam HTML digunakan untuk memberikan judul atau keterangan untuk sebuah grup elemen dalam tag fieldset. Ini membantu dalam memberikan konteks dan informasi tambahan tentang isi grup tersebut.

Element Datalist

Digunakan untuk menyediakan daftar opsi atau nilai yang dapat dipilih oleh pengguna dalam sebuah input teks. Ini sering digunakan dengan input teks untuk memberikan saran atau pilihan kepada pengguna saat mereka mengetik.
