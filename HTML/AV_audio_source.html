Element audio

Audio adalah elemen HTML yang digunakan untuk menyematkan file audio ke dalam halaman web. Dengan menggunakan elemen audio, Anda dapat memutar audio secara langsung di dalam browser tanpa perlu menggunakan pemutar audio eksternal.

Klik tombol putar untuk memutar suara:



Attribut audio

  • < audio controls autoplay>
    < source src="audio.mp3" type="audio/mpeg">
    < / audio>

Adalah atribut boolean yang, jika disertakan, akan memulai pemutaran audio secara otomatis begitu halaman dimuat, tanpa memerlukan interaksi pengguna tambahan.

Namun, penting untuk diingat bahwa penggunaan atribut autoplay bisa jadi mengganggu pengalaman pengguna terutama jika audio dimainkan secara tiba-tiba tanpa permintaan atau keinginan pengguna.

  • < audio controls>
    < source src="audio.mp3" type="audio/mpeg">
    < /audio>

Adalah atribut boolean yang, jika disertakan, akan menampilkan kontrol pemutar audio standar di browser untuk mengontrol pemutaran audio.

  • < audio controls loop>
    < source src="audio.mp3" type="audio/mpeg">
    < /audio>

Atribut loop pada elemen audio adalah atribut boolean yang, jika disertakan, akan membuat audio diputar berulang kali (loop) secara otomatis setelah selesai diputar satu kali. Dengan kata lain, audio akan terus diputar ulang tanpa henti sampai pengguna menghentikannya secara manual atau navigasi diubah.

  • < audio controls muted>
    < source src="audio.mp3" type="audio/mpeg">
    < /audio>

Adalah atribut boolean yang, jika disertakan, akan membisukan audio secara otomatis saat pemutaran dimulai. Dengan kata lain, audio tidak akan mengeluarkan suara sampai pengguna mengaktifkan kembali suaranya.

  • < audio controls preload="auto">
    < source src="audio.mp3" type="audio/mpeg">
    < /audio>

Adalah atribut yang mengontrol bagaimana browser memuat file audio sebelum pengguna mulai memutarnya. Atribut ini membantu mengatur keseimbangan antara penggunaan data/bandwidth dan pengalaman pengguna dalam hal seberapa cepat audio mulai diputar.

  1. none: Browser tidak akan memuat audio sebelum pengguna memulainya. Ini menghemat bandwidth dan data, tetapi mungkin menyebabkan penundaan saat pengguna memulai pemutaran.
  2. metadata: Browser hanya akan memuat metadata (seperti durasi dan ukuran file) dari audio. Ini memberikan sedikit informasi kepada pengguna tanpa memuat seluruh file.
  3. auto: Browser akan mencoba memuat seluruh audio, bahkan sebelum pengguna mulai memutarnya. Ini memastikan pemutaran cepat tetapi menggunakan lebih banyak bandwidth dan data.
  • < audio controls src="audio.mp3">< /audio>

Digunakan untuk menentukan sumber file audio yang akan diputar. Ini adalah salah satu cara untuk menyematkan file audio ke dalam halaman web. Saat menggunakan atribut src, Anda tidak perlu menggunakan elemen source yang terpisah.



Element source

Digunakan untuk menentukan beberapa sumber media untuk elemen multimedia seperti audio dan video. Elemen ini memungkinkan browser untuk memilih dan memutar sumber yang didukung oleh browser tersebut.

Tag source> dalam HTML digunakan untuk menentukan beberapa sumber media untuk elemen seperti video>, audio>, dan picture>. Tag ini sangat berguna untuk menyediakan berbagai format atau versi konten media agar kompatibel dengan berbagai browser dan perangkat.


Attribut source

  • < source media="(min-width:650px)" srcset="img_pink_flowers.jpg">

Digunakan untuk menentukan kondisi media tertentu di mana sumber ini akan digunakan. Ini memungkinkan Anda untuk memberikan berbagai sumber media berdasarkan kondisi seperti lebar layar, orientasi perangkat, resolusi, atau jenis perangkat.

dalam HTML digunakan untuk menentukan kondisi media query yang harus dipenuhi agar sumber media tersebut digunakan. Atribut ini biasanya digunakan dalam konteks elemen picture> untuk gambar responsif, tetapi juga dapat digunakan dengan elemen video> dan audio>.

Atribut media pada tag source> memungkinkan Anda untuk menyediakan sumber media yang lebih fleksibel dan responsif berdasarkan kondisi media query. Ini meningkatkan pengalaman pengguna dengan memastikan mereka mendapatkan versi konten yang paling sesuai dengan perangkat mereka.

  • < source src="horse.ogg" type="audio/ogg">

Atribut src pada tag source> dalam HTML digunakan untuk menentukan URL dari file media yang akan digunakan. Atribut ini sangat penting karena menunjuk langsung ke sumber media seperti video, audio, atau gambar yang ingin Anda sajikan dalam elemen video>, audio>, atau picture>.

  • < source srcset="image-large.jpg" media="(min-width: 800px)">

Atribut srcset pada tag source> dalam HTML digunakan untuk menentukan beberapa sumber gambar dengan berbagai resolusi dan ukuran. Atribut ini biasanya digunakan dalam konteks elemen picture> untuk mendukung gambar responsif, tetapi juga dapat digunakan langsung dalam elemen img>.

  • < source src="horse.ogg" type="audio/ogg">

dalam HTML digunakan untuk menentukan tipe MIME (Multipurpose Internet Mail Extensions) dari sumber media yang ditunjukkan oleh atribut src atau srcset. Menyediakan tipe MIME membantu browser untuk menentukan apakah mereka dapat memproses dan memutar file media yang ditentukan.

Dengan menggunakan atribut type, Anda dapat memastikan bahwa file media yang tepat dipilih dan digunakan oleh browser
