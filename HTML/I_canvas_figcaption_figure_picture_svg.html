Element canvas

canvas adalah elemen HTML yang menyediakan ruang gambar dinamis di halaman web. Ini memungkinkan Anda untuk membuat dan menggambar grafik, animasi, dan visualisasi lainnya secara dinamis menggunakan JavaScript.

adalah alat yang kuat untuk membuat gambar dinamis dan interaktif di halaman web Anda. Dengan menggabungkan elemen canvas dengan JavaScript, Anda dapat membuat berbagai jenis proyek kreatif dan interaktif yang memperkaya pengalaman pengguna Anda.





Attribut canvas

  • < canvas id="myCanvas2" width="200" height="200" style="border:1px solid">

Digunakan untuk menentukan tinggi dari area gambar di dalam elemen tersebut. Ini adalah atribut opsional yang dapat digunakan untuk menentukan ukuran vertikal dari elemen canvas.

  • < canvas id="myCanvas" width="800" height="600">

Digunakan untuk menentukan lebar dari area gambar di dalam elemen tersebut. Ini adalah atribut opsional yang digunakan untuk menentukan ukuran horizontal dari elemen canvas.



Element figcaption

figcaption adalah bagian dari elemen figure dan digunakan untuk menyediakan deskripsi atau keterangan untuk elemen figure tersebut. Elemen figure biasanya digunakan untuk mengelompokkan konten media (seperti gambar, ilustrasi, grafik, video, dan lainnya) bersama dengan keterangan yang sesuai.

Elemen figcaption dapat berisi teks atau elemen HTML lainnya, seperti paragraf, link, atau elemen lainnya yang diperlukan untuk keterangan yang sesuai.



Element figure

Digunakan untuk mengelompokkan konten media, seperti gambar, ilustrasi, diagram, audio, atau video, bersama dengan keterangan yang relevan yang disediakan menggunakan elemen < figcaption>.

< figure>
< img src="pic_trulli.jpg" alt="Trulli" style="width:100%">
< figcaption>Fig.1 - Trulli, Puglia, Italy.
< / figure >



Element picture

  • < picture > < source media="(min-width:650px)" srcset="img_pink_flowers.jpg"> < /picture>

Picture adalah elemen HTML yang digunakan untuk menyediakan sumber gambar alternatif dengan resolusi dan format yang berbeda untuk situasi tampilan yang berbeda. Ini memungkinkan pengembang web untuk memberikan gambar yang optimal sesuai dengan ukuran layar dan kemampuan perangkat pengguna.



Element svg

svg adalah elemen HTML yang digunakan untuk menambahkan grafik vektor ke halaman web. SVG (Scalable Vector Graphics) adalah format gambar XML yang mendefinisikan grafik dengan menggunakan metode vektor, yang berarti gambar-gambar tersebut dapat diperbesar atau diperkecil tanpa kehilangan kualitas.
