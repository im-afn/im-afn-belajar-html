Element label

< label for="username">Nama Pengguna:< /label>
< input type="text" id="username" name="username">

Digunakan untuk mendefinisikan label untuk elemen < input>, < textarea>, < select>, atau < button>. Ini adalah cara untuk menghubungkan teks dengan elemen formulir, membuat formulir lebih mudah diakses dan ramah pengguna.

Dalam contoh ini, elemen < label> "Nama Pengguna:" dihubungkan dengan elemen < input> menggunakan atribut for, yang sesuai dengan atribut id dari input. Ketika pengguna mengklik teks label, bidang input yang terkait akan fokus atau diaktifkan.



Attribut label

  • < label for="username">Username:< /label>

Digunakan untuk menunjukkan id dari elemen formulir yang terkait dengan label tersebut. Ini memungkinkan Anda untuk mengaitkan label dengan elemen formulir tertentu,

  • < input type="text" id="username" name="username" form="myForm">
