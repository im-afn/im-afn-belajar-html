Element select

Digunakan untuk membuat daftar dropdown dalam formulir web. Ini memungkinkan pengguna untuk memilih satu opsi dari daftar opsi. Elemen select biasanya digunakan bersama dengan elemen option untuk menentukan opsi yang tersedia dalam menu dropdown.



Attribut select

  • < select name="cars" autofocus>< /select>

Digunakan untuk menentukan bahwa sebuah dropdown list harus secara otomatis mendapatkan fokus saat halaman dimuat.

  • < select name="cars" disabled>< /select>

Digunakan untuk menonaktifkan sebuah dropdown list, yang berarti pengguna tidak dapat memilih atau berinteraksi dengan pilihan yang ada di dalamnya.

  • < select id="cars" name="carlist" form="carform">< /select>

Daftar turun bawah yang terletak di luar formulir (tetapi masih merupakan bagian dari formulir): Digunakan untuk menghubungkan elemen input ke formulir mana elemen tersebut akan dikirimkan ketika formulir tersebut



  • < select name="cars" multiple > select>

Digunakan untuk memungkinkan pengguna untuk memilih lebih dari satu opsi dari daftar dropdown. Ketika atribut multiple ditambahkan, pengguna dapat memilih beberapa opsi dengan menahan tombol Ctrl (di Windows) atau Command (di macOS) saat mengklik opsi-opsi yang diinginkan.

  • < select id="cars" name="cars">< /select>

Digunakan untuk memberikan nama atau identifikasi kepada elemen dropdown list



  • < select id="cars" name="cars" required> < /select>

Digunakan untuk menandai bahwa sebuah dropdown list harus dipilih sebelum formulir dapat disubmit.



  • < select name="cars" size="4">< /select>

Digunakan untuk menentukan jumlah baris yang akan ditampilkan dalam dropdown list secara default.



Element optgroup

digunakan dalam HTML untuk mengelompokkan opsi dalam elemen select. Ini memungkinkan Anda untuk membuat struktur hierarkis dalam elemen dropdown. Misalnya, jika Anda memiliki opsi yang terkait atau berkaitan satu sama lain, Anda dapat menempatkannya dalam grup yang sama dengan menggunakan tag optgroup.






Attribut optgroup

  • < optgroup label="German Cars" disabled>< /optgroup>

Digunakan untuk menonaktifkan seluruh grup opsi. Ini berarti pengguna tidak dapat memilih opsi apa pun dari grup yang dinonaktifkan tersebut.



  • < optgroup label="German Cars">< /optgroup>

Digunakan untuk memberikan label atau judul untuk grup opsi yang ditampilkan dalam elemen < select>. Label ini biasanya muncul di atas atau di depan grup opsi tersebut





Element Option

Digunakan dalam kombinasi dengan elemen select untuk membuat daftar pilihan dalam formulir web. Setiap tag option mewakili satu opsi dalam daftar.

Jika sebuah elemen select tidak memiliki elemen option di dalamnya, maka dropdown tersebut akan kosong dan pengguna tidak akan memiliki opsi untuk dipilih. Ini akan membuat elemen select tidak berfungsi dengan baik dalam formulir web.

Anda juga bisa menambahkan atribut selected ke salah satu tag



Attribut select

  • < option value="volvo" disabled>Volvo< /option>

Digunakan untuk menonaktifkan opsi tertentu dalam elemen select. Ketika sebuah opsi dinonaktifkan, pengguna tidak dapat memilihnya, dan opsi tersebut tidak akan ditampilkan sebagai pilihan yang dapat dipilih dalam dropdown.

  • < optgroup label="Swedish Cars">< /optgroup>

Sebenarnya tidak diperlukan, Namun jika Anda merasa perlu untuk menambahkan label secara eksplisit, Anda dapat menggunakan tag optgroup. Tag optgroup memiliki atribut label yang digunakan untuk memberikan label

  • < option value="saab" selected>Saab< /option>

Digunakan untuk menandai sebuah opsi sebagai pilihan awal (terpilih secara default) dalam elemen select. Ini memungkinkan Anda untuk menentukan opsi mana yang harus dipilih secara otomatis ketika halaman dimuat atau formulir ditampilkan.

  • < option value="volvo">Volvo< /option>

Digunakan untuk menentukan nilai yang akan dikirim ke server ketika pengguna memilih opsi tersebut. Nilai ini dapat berupa teks, angka, atau nilai lainnya yang akan digunakan untuk mengidentifikasi pilihan yang dipilih dalam formulir.
