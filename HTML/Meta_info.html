Element head

Elemen head> dalam HTML digunakan untuk menentukan informasi terkait dokumen HTML, seperti judul halaman, metadata, dan referensi ke berkas-berkas eksternal seperti CSS dan JavaScript

Element meta

Elemen meta> dalam HTML digunakan untuk menyediakan metadata atau informasi tambahan tentang dokumen HTML. Metadata ini tidak ditampilkan langsung di halaman web, tetapi digunakan oleh browser, mesin pencari, dan aplikasi lainnya untuk mengatur dan menafsirkan konten halaman.

  • < meta charset="UTF-8">: Menentukan pengkodean karakter yang digunakan dalam halaman.
  • < meta name="viewport" content="width=device-width, initial-scale=1.0">: Menentukan skala tampilan dan responsif pada perangkat seluler.
  • < meta name="description" content="Deskripsi singkat halaman web ini">: Memberikan deskripsi singkat tentang halaman web.
  • < meta name="keywords" content="kata kunci, yang, relevan">: Menentukan kata kunci yang relevan untuk halaman web.
  • < meta name="author" content="Nama Penulis">: Menentukan penulis atau pemilik halaman web.
  • < meta name="robots" content="index, follow">: Mengatur perilaku indeksasi mesin pencari terhadap halaman web.
  • < meta http-equiv="X-UA-Compatible" content="ie=edge">: Menentukan mode kompatibilitas IE.
  • < meta http-equiv="refresh" content="300">: Menentukan pembaruan otomatis halaman setiap 300 detik.

Attribut meta

  • < meta charset="UTF-8">

Atribut charset pada elemen meta> dalam HTML digunakan untuk menentukan pengkodean karakter (character encoding) yang digunakan dalam dokumen HTML. Pengkodean karakter menentukan cara karakter-karakter dalam dokumen diinterpretasikan oleh browser.

Pastikan untuk menggunakan pengkodean karakter yang sesuai dengan konten dokumen Anda untuk memastikan tampilan yang benar di berbagai platform dan perangkat.

  • < meta name="description" content="Ini adalah deskripsi singkat tentang halaman web ini">

Atribut content pada elemen meta> dalam HTML digunakan untuk menyediakan nilai atau konten tambahan untuk metadata yang ditentukan dalam elemen tersebut.

Metadata dalam bentuk meta> memberikan informasi tambahan tentang halaman web kepada browser, mesin pencari, dan aplikasi lainnya.

  • < meta http-equiv="refresh" content="5; URL=https://www.contoh.com">

Atribut http-equiv pada elemen meta> dalam HTML digunakan untuk menentukan instruksi tambahan yang berkaitan dengan protokol HTTP. Ini biasanya digunakan untuk memberikan informasi tentang metode yang diharapkan untuk diambil oleh browser atau server web dalam memproses halaman.

  • < meta name="description" content="Ini adalah deskripsi singkat tentang halaman web ini">

Atribut name pada elemen meta> dalam HTML digunakan untuk menentukan nama dari metadata yang akan ditentukan. Metadata ini memberikan informasi tambahan tentang halaman web kepada browser, mesin pencari, dan aplikasi lainnya.

Element base

Elemen base> dalam HTML digunakan untuk menentukan URL dasar untuk semua URL yang ditentukan dalam dokumen. Ini biasanya digunakan untuk menetapkan URL dasar untuk semua hyperlink dan URL yang terkait dalam halaman.

< base href="https://www.contoh.com/">

Elemen base> dalam HTML berfungsi seperti pengaturan URL dasar untuk semua tautan dalam halaman web. Bayangkan Anda memiliki banyak tautan di halaman web Anda, dan Anda ingin semuanya mengarah ke lokasi yang sama di situs web Anda. Alih-alih menentukan URL lengkap untuk setiap tautan, Anda bisa menggunakan elemen base> untuk menetapkan URL dasar, dan semua tautan akan dianggap relatif terhadap URL dasar tersebut.

Attribut base

  • < base href="https://www.example.com/">

Atribut href pada elemen base> dalam HTML digunakan untuk menetapkan URL dasar untuk semua tautan yang terkait dalam halaman. Dengan menetapkan URL dasar menggunakan atribut href, semua tautan dalam halaman akan dianggap relatif terhadap URL dasar yang ditetapkan.

  • < base href="https://www.contoh.com/" target="_blank">

Atribut target pada elemen base> memang ada dan digunakan untuk menentukan di mana semua tautan yang tidak memiliki atribut target spesifik akan dibuka. Ini berguna jika Anda ingin mengatur secara default bagaimana semua tautan di halaman Anda dibuka.

_self: (Default) Membuka tautan di frame yang sama atau tab yang sama.
_blank: Membuka tautan di tab baru atau jendela baru.
_parent: Membuka tautan di frame induk jika ada.
_top: Membuka tautan di seluruh jendela, menghapus semua frame.
