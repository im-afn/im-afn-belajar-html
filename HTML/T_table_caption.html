Element table

Elemen table> dalam HTML digunakan untuk membuat tabel yang terdiri dari baris (tr>) dan sel (td> untuk sel data atau th> untuk sel header).

Elemen table> adalah bagian inti dalam pembuatan tabel dalam HTML.

ID Nama Posisi
001 John Doe Manajer
002 Jane Smith Pengembang
Total Karyawan: 2

Element caption

Elemen caption> dalam HTML digunakan untuk menambahkan judul atau keterangan pada sebuah tabel. Elemen ini harus ditempatkan sebagai elemen pertama dalam sebuah tabel setelah elemen pembuka table>.

Dalam contoh ini, elemen caption> digunakan untuk menambahkan judul "Data Produk" pada tabel. Ini membantu dalam memberikan konteks atau informasi tambahan tentang isi tabel kepada pengguna.

Data Produk
Nama Harga
Mouse $10
Keyboard $20
Total Produk: 2
