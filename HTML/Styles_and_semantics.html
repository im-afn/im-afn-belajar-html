Element style

Tag style> dalam HTML digunakan untuk menyisipkan kode CSS langsung di dalam dokumen HTML. CSS yang ditulis di dalam tag style> hanya akan mempengaruhi elemen-elemen dalam dokumen tersebut.

Pengujian Cepat: Ideal untuk pengujian cepat atau saat hanya ada beberapa aturan CSS yang perlu diterapkan.

Spesifikasi Lokal: Berguna untuk menerapkan gaya khusus pada halaman tertentu tanpa mempengaruhi halaman lainnya.

Kemudahan Penggunaan: Memungkinkan penulisan CSS langsung dalam file HTML tanpa perlu membuat file CSS terpisah.

Tag style> sangat berguna untuk menyisipkan aturan CSS langsung ke dalam dokumen HTML. Ini memungkinkan fleksibilitas dalam mengatur tampilan halaman secara cepat dan efisien. Namun, untuk proyek besar atau kompleks, disarankan menggunakan file CSS eksternal untuk manajemen gaya yang lebih baik dan terstruktur.

Attribut style

  • @media

Atribut media pada tag style> digunakan untuk menentukan jenis perangkat atau kondisi tertentu di mana aturan CSS yang ada di dalam tag style> akan diterapkan. Atribut ini memungkinkan pengembang web untuk menargetkan perangkat tertentu atau kondisi tampilan tertentu (seperti ukuran layar) langsung dari tag style>.

  • style> type="text/css"> < /style>

Atribut type pada tag style> digunakan untuk menentukan jenis bahasa gaya yang digunakan di dalam tag style>. Dalam HTML5, nilai yang paling umum dan default untuk atribut type adalah text/css, yang menunjukkan bahwa bahasa gaya yang digunakan adalah CSS (Cascading Style Sheets).

Meskipun atribut type pada tag style> secara teknis tidak diperlukan dalam HTML5 karena nilai defaultnya adalah text/css, Anda tetap dapat menggunakannya untuk kejelasan dan keterbacaan kode. Jika Anda menulis gaya dalam format lain atau bekerja dalam lingkungan yang memerlukan kepatuhan ketat terhadap standar lama, menyertakan atribut type adalah praktik yang baik.

Element span

Tag span> dalam HTML adalah elemen inline yang serbaguna yang digunakan untuk mengelompokkan dan menggaya elemen-elemen inline. Berbeda dengan elemen blok seperti div>, yang membuat baris baru dan mengambil lebar penuh yang tersedia, span> tidak membuat baris baru dan hanya mengambil lebar sesuai dengan kontennya.
